## IoT Sensor Dashboard

This project demonstrates data collection from a simulated IoT temperature sensor,
data storage using SQLite, and visualization through a Python-based dashboard.

---

## Project Overview
The system simulates an IoT temperature sensor that periodically generates data.
The collected data is stored in a database and visualized as a time-series dashboard.
